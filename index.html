About

M8 Bot is a Discord bot that allows Mixer streamers to have their streams announced in their Discord servers.

In addition to Mixer streams, M8 Bot has a plethora of other commands to spice up your discord server.

Setup

Setting up M8 Bot is very easy and quick to do!

Requirements:

  • Have a discord account
  • Have a discord server (or have admin permissions on one)
  • Some kind of web browser

    • Step 1: Invite

  • Click here to invite the bot to your server.
  • Pick your server from the list, and click "Authorize".

    • Step 2: Test

  • In order to test the bot, simply type "!ping" into any chat where the bot can see it, and reply.
  • If the bot replies to your message, all is good!

    • Step 3: Setting Up Streamers (Optional)

  • Navigate to the chat channel where you want the bot to announce the streamer(s).
  • In the chat, type "!add-streamer" followed by the streamer's Mixer name. For example, "!add-streamer ComixsYT".
  • After, type "!mystreamers" to get a list of the streamers assigned to that chat channel. If the new streamer is there, you are done.

    • Step 3.5: Removing Streamers (Optional)

  • Navagate to the chat channel where you want to stop the accouncements.
  • In the chat, type "!del-streamer" followed by the streamer's Mixer name. For example, "!del-streamer ComixsYT".
  • After, type "!mystreamers" to get a list of the streamers assigned to that chat channel. If the streamer is no longer there, you are done.

    • Commands

    In addition to stream announcements, M8 Bot has some commands that are sure to spruce up your discord!

    Help for M8 Bot as of 6/14/17

    • !help m8bot - Sends this help Message.
    • !ping - Used to test if the bot is listening. A response should appear and then disappear along with the command.
    • !m8status - Sends a status report of the bot.
    • !me - Sends you information about you.
    • !pun/!dadjoke - Sends a funny pun courtesy of murfGUY's dadjoke database.
    • !cn/!chuck - Pulls a random Chuck Norris fact.
    • !blamecomixs - Used whenever ComixsYT does something stupid and must be blamed!
    • !unblamecomixs - Used whenever ComixsYT does something good and must be unblamed!
    • !lenny - ( ͡° ͜ʖ ͡°)
    • !ascii - Gets whatever you typed after !ascii and makes it into ASCII art.
    • !urban/!define - Get the urban definition of a word.
    • !lmgthf/!google - Gets a Let Me Google That For You link for any term you want.
    • !mfinger - Allows you to give the finger that is located in the middle of our appendages to others.
    • !fail - For when someone seriously fails.
    • !pepe - Who doesn’t love Pepe?
    • !tank - GTFO of my way I got a fucking tank!
    • !hug/!hugs - Wanna give someone a hug? Do it then!
    • !m8bug - Returns a link to report bugs.
    • !add-streamer/!del-streamer - Used to add or delete streamers from that chat channel. Can only be run by the server owner or anyone with the "ADMINISTRATOR" permission.
    • !allstreamers - Lists all the streamers that the bot stalks.
    • !mystreamers - Lists all the streamers registered to that chat channel.
    • !mixer - Gets info about a mixer user. Usage - !mixer NAME
    • !ping-on/!ping-off - Allows a server owner/admin to decide whether or not M8 Bot can use @here in that channel. The default is on.
    • !insult @name - Used to insult another user.
    • !steam ___ - Used get info on a steam game.
    • !coin - Flips a coin.
    • !rawr - I'm a meme.
