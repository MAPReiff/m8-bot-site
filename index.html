About

M8 Bot is a Discord bot that allows Twitch & Mixer streamers to have their streams announced in their Discord servers. In addition to Twitch & Mixer streams, M8 Bot has a plethora of other commands to spice up your discord server.

Setup

Requirements:

  • Have a discord account
  • Have a discord server (or have admin permission on one)
  • Some kind of web browser

Step 1:

  • Click here to invite the bot to your server
  • Pick your server from the list, and click "Authorize".

Step 2:

  • In order to test the bot, simply type "!ping" into any chat where the bot can see it, and reply.
  • If the bot replies to your message, all is good!
    • If not, please make sure the bot has proper permissions to chat!

Step 3: Configuring Custom Options (Optional)

  • At the current moment, the available options are:
    • prefix - the prefix for all M8 Bot commands in your server. The default is "!"
    • modLogChannel - the channel where moderation actions are logged. The default is "mod-log"
    • modRole - the role that is allowed to run low-level mod commands. The default is "Moderator"
    • adminRole - the role that is allowed to run high-level mod commands. The default is "Administrator"
    • systemNotice - this gives a notice when a user tries to run a command that they do not have permission to use. The default is "true"
    • welcomeChannel - the channel where welcome messages are sent. Default is "welcome"
    • welcomeMessage - the message sent in the welcome channel when a new user joins. The default is "Welcome {{user}} to the {{serverName}} server! There are now {{memberCount}} members in this server. Enjoy your stay."
      • {{user}} is automaticaly replaced with the new member's name.
      • {{serverName}} is automaticaly replaced with the server's name.
      • {{memberCount}} is automaticaly replaced with the number of users now in the server.
      • Setting the welcomeMessage to "Welcome IMG" (without quotes) will post an image like this instead of a message.
    • welcomeEnabled - whether or not the welcomeMessage is enabled. Default is "false"
    • defaultRole - the default role users are assigned when they join the server. Default is "Member"
    • defaultRoleEnabled - whether or not the defaultRole is enabled. Default is "false"
    • bannedWords - list of banned words, separated with ", ". Default is "Bad_word_here"
      • For example, if you want to ban "wow", "cool" and "Zamasu did nothing wrong", the list would look like "wow, cool, Zamasu did nothing wrong".
      • People with the adminRole can bypass this ban.
  • In order to view the settings, run !set
  • In order to change the settings, run !.set edit <setting> <option>, replacing <setting> with the setting name, and <option> with the new option.
  • Settings that are true or false by default must always be either true or false.
  • Settings that ask for a channel name must be given the channel name without the # symbol, just the name.
  • Settings that ask for a role name must be given the role name without the @ symbol, just the name.

Everything from now on in this guide will use the "!" prefix. If you changed it, replace "!" with your custom prefix!

Step 4a: Setting Up Mixer Streamers (Optional)

  • Navigate to the chat channel where you want the bot to announce the Mixer streamer(s).
  • In the chat, type "!addmixer" followed by the streamer's Mixer name. For example, "!addmixer ComixsYT".

    • Step 4b: Setting Up Twitch Streamers (Optional)

  • Navigate to the chat channel where you want the bot to announce the Twitch streamer(s).
  • In the chat, type "!addtwitch" followed by the streamer's Twitch name. For example, "!addtwitch ComixsYT".

    • Step 4.5a: Removing Mixer Streamers (Optional)

  • Navagate to the chat channel where you want to stop the accouncements.
  • In the chat, type "!delmixer" followed by the streamer's Mixer name. For example, "!delmixer ComixsYT".

    • Step 4.5b: Removing Twitch Streamers (Optional)

  • Navagate to the chat channel where you want to stop the accouncements.
  • In the chat, type "!deltwitch" followed by the streamer's Twitch name. For example, "!deltwitch ComixsYT".

    Help for M8 Bot as of 3/4/18

    General Bot:
    • !help - Sends this help Message.
    • !ping - Used to test if the bot is listening. A response should appear and then disappear along with the command.
    • !m8status - Sends a status report of the bot.
    • !config - Allows server owners to view their configuration.
    • !config edit - Allows server owners to edit their configuration.
    • !m8bug - Returns a link to report bugs.
    • !changelog - Bot changelog.
    Moderation:
    • !purge # - Deletes the last # of messages up to 100 at a time.
    • !warn [@person] [reason] - Sends a warning message to a rule breaker. Saves a copy to the configured mod log channel. Configured Moderator rank and up.
    • !kick [@person] [reason] - Kicks the rule breaker from the server. Saves a copy to the configured mod log channel. Configured Moderator rank and up.
    • !ban [@person] [reason] - Bans the rule breaker from the server. Saves a copy to the configured mod log channel. Configured Administrator rank and up.
    Live Streams:
    • !addmixer or !delmixer - Used to add or delete Mixer streamers from that chat channel. Can only be run by the server owner or anyone with the "ADMINISTRATOR" permission.
    • !addtwitch or !deltwitch - Used to add or delete Twitch streamers from that chat channel. Can only be run by the server owner or anyone with the "ADMINISTRATOR" permission.
    • !mystreamers - Lists all the streamers registered to that chat channel.
    Information:
    • !me - Sends you information about you.
    • !mixer - Gets info about a mixer user. Usage - !mixer NAME
    Economy:
    • !balance or !>bal - Gets your current balance of Glix and cooldown timers.
    • !work - Work for Glix. Uses the Earnings Cooldown (30 min)
    • !crime - Commit crimes for Glix. Higher payout than work, but a high risk of it failing. Uses the Earnings Cooldown (30 min)
    • !slots - Try your luck at the M8 Bot Casino. Users the Games Cooldown (30 sec)
    • !bail or !earn-again - Resets your Earnings Cooldown for the cost of 5 Glix

    • !urban or !define - Get the urban definition of a word.

    Zamasu

    Zamasu is a discord bot strictly designed around moderation. It is based on the Dragon Ball Super character of the same name. In the show, Zamasu is a God who despises mortals, as well as their existence. I found him to be the perfect character to enforce rules and punish rule breakers. All of Zamasu's moderation functions can be found in M8 Bot, however, Zamasu will receive new moderation features first.
