About

M8 Bot is a Discord bot that allows Twitch & Mixer streamers to have their streams announced in their Discord servers. In addition to Twitch & Mixer streams, M8 Bot has a plethora of other commands to spice up your discord server.

Setup

Setting up M8 Bot is very easy and quick to do!

Requirements:

  • Have a discord account
  • Have a discord server (or have admin permissions on one)
  • Some kind of web browser

    • Step 1: Invite

  • Click here to invite the bot to your server.
  • Pick your server from the list, and click "Authorize".

    • Step 2: Test

  • In order to test the bot, simply type "!ping" into any chat where the bot can see it, and reply.
  • If the bot replies to your message, all is good!

    • Step 3: Configuring Custom Options (Optional)

  • At the current moment, the available options are - prefix, mod log channel, mod role, admin role, system notice, welcome channel, welcome message, welcome messagetoggle, default role & default role toggle.
  • Navigate to a chat channel where the bot cna read and send messages.
  • In the chat, type "!config" to see your current server settings. To edit a setting, type "!config edit NAME value", replacing NAME with the name of the option (such as prefix or welcomeChannel), and replacing value with the new setting (such as the new prefix).

    • Everything from now on in this guide will use the "!" prefix. If you changed it in step 3, replace "!" with your custom prefix!

    Step 4a: Setting Up Mixer Streamers (Optional)

  • Navigate to the chat channel where you want the bot to announce the Mixer streamer(s).
  • In the chat, type "!add-streamer-mixer" followed by the streamer's Mixer name. For example, "!add-streamer-mixer ComixsYT".

    • Step 4b: Setting Up Twitch Streamers (Optional)

  • Navigate to the chat channel where you want the bot to announce the Twitch streamer(s).
  • In the chat, type "!add-streamer-twitch" followed by the streamer's Twitch name. For example, "!add-streamer-twitch ComixsYT".

    • Step 4.5a: Removing Mixer Streamers (Optional)

  • Navagate to the chat channel where you want to stop the accouncements.
  • In the chat, type "!del-streamer-mixer" followed by the streamer's Mixer name. For example, "!del-streamer-mixer ComixsYT".

    • Step 4.5b: Removing Twitch Streamers (Optional)

  • Navagate to the chat channel where you want to stop the accouncements.
  • In the chat, type "!del-streamer-twitch" followed by the streamer's Twitch name. For example, "!del-streamer-twitch ComixsYT".

    • Commands

    In addition to stream announcements, M8 Bot has some commands that are sure to spruce up your discord! Commands here are shown with the default prefix!

    Help for M8 Bot as of 1/15/18

    • !help m8bot - Sends this help Message.
    • !ping - Used to test if the bot is listening. A response should appear and then disappear along with the command.
    • !m8status - Sends a status report of the bot.
    • !config - Allows server owners to view their configuration.
    • !config edit - Allows server owners to edit their configuration.
    • !purge # - Delets the last # of messages up to 100 at a time.
    • !warn [@person] [reason] - Sends a warning message to a rule breaker. Saves a copy to the configured mod log channel. Configured Moderator rank and up.
    • !kick [@person] [reason] - Kicks the rule breaker from the server. Saves a copy to the configured mod log channel. Configured Moderator rank and up.
    • !ban [@person] [reason] - Bans the rule breaker from the server. Saves a copy to the configured mod log channel. Configured Administrator rank and up.
    • !me - Sends you information about you.
    • !pun/!dadjoke - Sends a funny pun courtesy of murfGUY's dadjoke database.
    • !cn/!chuck - Pulls a random Chuck Norris fact.
    • !lenny - ( ͡° ͜ʖ ͡°)
    • !ascii - Gets whatever you typed after !ascii and makes it into ASCII art.
    • !urban/!define - Get the urban definition of a word.
    • !lmgthf/!google - Gets a Let Me Google That For You link for any term you want.
    • !hug/!hugs - Wanna give someone a hug? Do it then!
    • !spank - Wanna give someone a spanking? Do it then!
    • !m8bug - Returns a link to report bugs.
    • !add-streamer-mixer/!del-streamer-mixer - Used to add or delete Mixer streamers from that chat channel. Can only be run by the server owner or anyone with the "ADMINISTRATOR" permission.
    • !add-streamer-twitch/!del-streamer-twitch - Used to add or delete Twitch streamers from that chat channel. Can only be run by the server owner or anyone with the "ADMINISTRATOR" permission.
    • !mystreamers - Lists all the streamers registered to that chat channel.
    • !mixer - Gets info about a mixer user. Usage - !mixer NAME
    • !ping-on/!ping-off - Allows a server owner/admin to decide whether or not M8 Bot can use @here in that channel. The default is on.
    • !steam ___ - Used get info on a steam game.
    • !coin - Flips a coin.
    • !rawr - I'm a meme.
